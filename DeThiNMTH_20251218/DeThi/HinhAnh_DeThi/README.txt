THƯ MỤC HÌNH ẢNH MẪU CHO ĐỀ THI
================================

Thư mục này chứa tất cả hình ảnh, biểu đồ, icon mẫu để sinh viên sử dụng trong đề thi.

==========================================
PHẦN 1: MICROSOFT WORD
==========================================

1. Logo_CongTy_ABC.png / Logo_CongTy_ABC.svg
   - Dùng cho: Hình 1 - Logo công ty
   - Kích thước yêu cầu: 6cm × 4cm
   - Vị trí: Căn trái, text wrapping: Square
   - Caption: "Hình 1: Logo công ty"

2. BieuDo_DoanhThu.png / BieuDo_DoanhThu.svg
   - Dùng cho: Hình 2 - Biểu đồ doanh thu
   - Kích thước yêu cầu: 8cm × 5cm
   - Vị trí: Căn phải, text wrapping: Tight
   - Caption: "Hình 2: Biểu đồ doanh thu"

3. Icon_MuiTen_Len.png
   - Dùng cho: Cột "Biểu đồ" trong Bảng 1 (Q2, Q3, Q4 - tăng trưởng)
   - Màu: Xanh (#2E75B6)
   - Kích thước: Nhỏ, vừa với ô bảng

4. Icon_MuiTen_Xuong.png
   - Dùng cho: Cột "Biểu đồ" trong Bảng 1 (nếu có giảm)
   - Màu: Đỏ (#E74C3C)
   - Kích thước: Nhỏ, vừa với ô bảng

5. Icon_TangTruong.png
   - Dùng cho: Cột "Biểu đồ" trong Bảng 1 (biểu tượng tăng trưởng)
   - Màu: Xanh lá (#27AE60)
   - Kích thước: Nhỏ, vừa với ô bảng

==========================================
PHẦN 2: MICROSOFT EXCEL
==========================================

LƯU Ý: Các biểu đồ trong Excel sẽ được tạo trực tiếp từ dữ liệu trong Sheet "BanHang".
Không cần hình ảnh mẫu cho phần Excel vì sinh viên sẽ tự tạo biểu đồ từ dữ liệu.

Tuy nhiên, sinh viên có thể tham khảo:
- BieuDo_DoanhThu.png: Để biết cách trình bày biểu đồ cột
- Các icon có thể được sử dụng trong Conditional Formatting (Icon Sets)

==========================================
HƯỚNG DẪN SỬ DỤNG
==========================================

1. Mở thư mục HinhAnh_DeThi
2. Chọn file hình ảnh phù hợp theo yêu cầu trong đề thi
3. Chèn vào tài liệu Word/Excel/PowerPoint
4. Điều chỉnh kích thước và định dạng theo yêu cầu cụ thể

LƯU Ý:
------
- Có thể sử dụng file PNG hoặc SVG
- Nếu file SVG không hiển thị được, hãy sử dụng file PNG
- Có thể mở file SVG bằng trình duyệt web và chụp màn hình để chuyển sang PNG nếu cần
- Tất cả hình ảnh đều có thể resize mà không mất chất lượng (PNG) hoặc vector (SVG)
